# Security Policy

## Supported Versions

iRaceHub is currently maintained on the default branch. Security fixes are applied there first.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report them privately to:

- GitHub: https://github.com/DarrellRichards

When reporting an issue, include:

- a description of the vulnerability
- steps to reproduce it
- possible impact
- any suggested remediation if you have one

You should receive an acknowledgment as soon as practical. After triage, fixes
will be prepared and released as appropriate.
